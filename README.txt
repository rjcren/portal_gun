本人初学者，mod正在做
本mod基于我的世界1.18.1和forge39.1.0制作

在外部库  net.minecraft:client:extra:版本号
    /assets/minecraft 中可以看到: 游戏所有的材质（textures），方块状态（blockstates），翻译文件（lang），粒子效果（particles）等
    /data/minecraft   中可以看到: 合成台、冶炼配方（recipes），战利品表（loot_tables）

在外部库  net.minecraftforge:forge:forge版本号_mapped_official_版本号/net中可以看到所有事件类及其方法
